David, an HR manager, maintains a dynamic list of employee IDs using a linked list. Occasionally, he needs to remove a specific employee from the list based on their position in the sequence. Implement a function using circular linked list that deletes the k-th employee from the list and returns the updated list.

Input format :
The first line of input consists of an integer (n), representing the number of employees in the linked list.

The second line consists of n space-separated integers, representing the employee IDs in the linked list.

The third line consists of an integer (k), representing the 1-based position of the employee to remove.

Output format :
The program outputs the updated list of employee IDs after deleting the employee at position k, printed as space-separated integers in a single line.



Refer to the sample output for the formatting specifications.

Code constraints :
In this scenario, the given test cases will fall under the following constraints:

1 ≤ n ≤ 150

1 ≤ Employee ID ≤ 104

1 ≤ k ≤ n

Sample test cases :
Input 1 :
5
10 20 30 40 50
4
Output 1 :
10 20 30 50

7

import java.util.Scanner;

  class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // Input 1: Size of array
        int n = sc.nextInt();
        int[] arr = new int[n];

        // Input 2: Elements
        for (int i = 0; i < n; i++) {
            arr[i] = sc.nextInt();
        }

        // Input 3: Value to remove
        int target = sc.nextInt();

        // Logic: Print everything except the target value
        for (int i = 0; i < n; i++) {
            if (i!=target-1) {
                System.out.print(arr[i] + " ");
            }
        }
    }
}
